Chapter 1: What is HTML?

HTML is an acronym that stands for Hypertext Markup Language. In fact, this website was made from HTML! (Really!) The person who invented HTML is... Tim Berners-Lee!

First, let's get on the topic of Markup Languages. A markup language means that in order to have elements to make up the webpage, we need to have an opening tag and a closing tag. For example to write a paragraph, you would need to type for the opening tag and for the closing tag.
