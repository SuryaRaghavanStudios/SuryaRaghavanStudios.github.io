Setting Up

Before we write real code, we need to setup! What that means is that we need to have an proper IDE or Intergrated Development Environment. That means that we need to have a good text or code editor and a browser to work with. In this website we will be using a free code editor called Visual Studio Code. Of course you can use another text or code editor to work with! Below is all the download links for good code editors. Or you can use Notepad (built-in to windows) or TextEdit (built-in to Mac)

on the left hand side menu. Search for "live server" and click on the first result you see. Click on "Install" at the right hand side. It is green.
